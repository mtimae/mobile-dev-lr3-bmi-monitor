# ЛР3 — Мониторинг веса и ИМТ

## Описание проекта
Мобильное Android-приложение **«Мониторинг веса и ИМТ»**, разработанное в рамках лабораторной работы по дисциплине **«Разработка мобильных приложений»**.

Приложение предназначено для расчёта индекса массы тела (ИМТ), определения категории массы тела пользователя и сохранения результатов в истории измерений.

## Тема лабораторной работы
**Медицинское приложение**

## Вариант
**Вариант 24 — «Мониторинг веса и ИМТ»**

## Используемые технологии
- Kotlin
- Android Studio
- XML-разметка интерфейса
- ViewBinding
- SharedPreferences

## Функциональные возможности
- ввод роста пользователя в сантиметрах;
- ввод веса пользователя в килограммах;
- расчёт индекса массы тела;
- определение категории массы тела;
- сохранение результата в историю измерений;
- отображение истории ранее сохранённых записей;
- восстановление истории после повторного запуска приложения.

## Структура проекта
- `MainActivity.kt` — основная логика приложения;
- `activity_main.xml` — главный экран приложения;
- `AndroidManifest.xml` — конфигурация приложения;
- `strings.xml` — строковые ресурсы интерфейса.

## Инструкция по запуску
1. Установить Android Studio.
2. Открыть проект.
3. Подключить Android-смартфон по USB.
4. Включить режим разработчика и USB-отладку на телефоне.
5. Выбрать устройство в Android Studio.
6. Нажать **Run**.

## Инструкция по использованию
1. Запустить приложение на смартфоне.
2. Ввести рост в сантиметрах.
3. Ввести вес в килограммах.
4. Нажать кнопку **«Рассчитать ИМТ»**.
5. Ознакомиться с рассчитанным значением ИМТ и категорией массы тела.
6. Нажать кнопку **«Сохранить запись»** для добавления результата в историю измерений.

## Результат
Разработано мобильное приложение для мониторинга веса и ИМТ, позволяющее рассчитывать индекс массы тела, определять категорию массы тела и сохранять историю измерений между запусками.
